幂平均不等式
----------

幂平均不等式的图像示例, 这里仅考虑二元情况.

一组正数 :math:`a_1,a_2,\cdots,a_n` 的 :math:`x` 次幂平均定义为

.. math ::

    M_x(a_1,a_2,\cdots,a_n) = \left(\cfrac{1}{n}\sum_{i=1}{n}{a_i^x}\right)^{\cfrac{1}{x}}.

其中当 :math:`x=0` 时, 补充定义

.. math ::

    M_0(a_1,a_2,\cdots,a_n) = (a_1a_2\cdots a_n)^{\cfrac{1}{n}}.

可以证明这个补定义使其在实数域上连续.
